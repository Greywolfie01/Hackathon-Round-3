About the Application

This full-stack web application is a game-changer for J's Cupcake Company. Designed with an intuitive user interface, it serves as a centralized platform for all business operations.

The application provides Jodi, the founder, with comprehensive dashboards for monitoring financial transactions, inventory, and customer interactions.

All transactions are recorded in South African Rand (ZAR), in line with the company's financial guidelines.

The application is hosted on Google Cloud, ensuring high availability and security.
